Use freeCodeCamp to learn the below concepts

  • Use Hex Code for Specific Colors e.g. #000000 - uses hex values (0-15 for ea value) to represent and mix red, green, and blue (over 16 million colors)
  • Use Abbreviated Hex Code e.g. #fff - uses hex values (0-15 for ea value - 1 digit ea) to represent and mix red, green, and blue (~ 4,000)
  • Use RGB values to Color Elementse.g. rgb(0,11,10) works just like hex except uses decimal values 0 - 255 (16x16 = 256, so also allows over 16 million colors)
